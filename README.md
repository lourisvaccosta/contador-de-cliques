# 🖱️ Contador de Cliques Básico em React

Este é um projeto introdutório desenvolvido em React para colocar em prática os fundamentos da biblioteca. A aplicação consiste em um contador de cliques simples que atualiza a interface dinamicamente a cada interação do usuário.

## 🎯 Objetivo do Projeto

O principal objetivo deste desafio foi entender e aplicar na prática três conceitos essenciais do React:
1. **Componentização:** Criação de componentes funcionais com responsabilidade única.
2. **Gerenciamento de Estado:** Utilização de Hooks para armazenar e atualizar dados.
3. **Manipulação de Eventos:** Captura de interações do usuário na interface.

## ✨ Funcionalidades

* Incremento de um contador numérico a cada clique no botão.
* Atualização da interface em tempo real, refletindo o valor exato do estado atual.



## 🚀 Tecnologias e Conceitos Utilizados

* **React:** Biblioteca JavaScript para construção da interface de usuário.
* **JSX:** Sintaxe que mistura HTML e JavaScript para renderizar os elementos na tela.
* **React Hooks (`useState`):** O "coração" da aplicação, responsável por memorizar o número atual de cliques e avisar o React para atualizar a tela quando esse número muda.
* **Eventos (`onClick`):** Escuta de eventos de clique aplicados diretamente no botão.
